# ■ サービス概要 
  ウイスキー好きな方はもちろん、ウイスキー初心者の方にもウイスキーの歴史、種類、楽しみ方を知ってもらうことで、ウイスキーに関心を持ってもらい、
  また、その関心をウイスキー好きたちで 情報交換し合うことで、自分に合うウイスキーを導き出すためのサービスです。

# ■ メインのターゲットユーザー 
  20代〜60代の男女(現在ウイスキーを楽しんでいる方、これからウイスキーを楽しみたい方)

# ■ ユーザーが抱える課題 
  - これからウイスキーを趣味としたいが、少し敷居が高く感じており、また知識がないため、何を飲んだらいいのかが分からない人。 
  - 長年ウイスキーを楽しんできたが、他のお酒に比べると情報量が少ないため、より情報共有したい人。

# ■ 解決方法 
  - 豆知識機能を通じて、ウイスキーの歴史、特徴、種類、飲み方、おすすめを知ることで、知識を付けてもらう。
  - ショップ検索機能を利用して、酒屋の位置情報の取得および実際にウイスキーを購入してもらう。
  - 投稿機能にて、自分が飲んだウイスキーのレビュー・感想を発信してもらう。
  - コメント機能を通じて、他ユーザーとの交流を図り、ウイスキーの関心をより深めてもらう。

# ■ なぜこのサービスを作りたいのか？ 
  お酒の中でも「ウイスキー」と言われると、「お高いイメージ」「アルコール感が強い」など、様々なイメージがあるかと思います。 現在、ウイスキーはスーパーマーケットなどの一般小売店での取り扱いが非常に少なく、専門の酒屋に行かないと一定数の取り扱いがない状況であります。 また、ウイスキーの情報量は他のお酒類と比べると比較的少ないため、ウイスキー初心者からすれば、どうしても敷居が高く感じてしまい、結局、どんなウイスキーがあるのか？どのウイスキーを買えばいいのか？など分からないことが多すぎるが故に、他のお酒に流れてしまいます。ただ私は、ウイスキー好きであった祖父からウイスキーの豆知識を聞きながら一緒に飲んでいるうちに、ウイスキーの種類や製法、ウイスキーの特色（飲みやすさ等）、ウイスキーの様々な飲み方などを学ぶことができました。その些細な豆知識を少しでも知ることで、ウイスキーに対する見方が大きく変わると考えております。上記の現状および自身の体験から、私は、より多くの方（特に同年代）にもっと身近に！手軽に！ウイスキーを楽しんでほしいと思い、このサービスを作ろうと思いました。 このサービスを機会に、ウイスキーの知識を少しでも身に付けてもらうことで、ウイスキーの素晴らしさ、奥深さを知っていただき、また、サービスを利用する全世代のウイスキー好きの方々が交流することで、 ご自身の Whiskey Life を見つけてほしいと願っております。
